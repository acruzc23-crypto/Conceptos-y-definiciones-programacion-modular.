# Manipulación de Vectores con JavaScript

## Introducción

El presente proyecto tiene como finalidad ampliar las funcionalidades de manipulación de vectores mediante la incorporación de nuevas herramientas que permiten realizar operaciones avanzadas sobre los datos almacenados. Para ello, se implementaron diversos algoritmos de ordenamiento, búsqueda y transformación, los cuales facilitan la organización, localización y modificación de los elementos contenidos en un vector numérico.

Entre las funcionalidades desarrolladas se encuentran los métodos de ordenamiento por selección, burbuja e inserción, así como los algoritmos de búsqueda secuencial y búsqueda binaria. Además, se incorporaron opciones de transformación que permiten invertir el orden de los elementos, eliminar valores duplicados, rotar posiciones dentro del vector e intercambiar elementos específicos. Estas mejoras fortalecen la capacidad del sistema para gestionar información de manera más eficiente y dinámica.

## Objetivos

### Objetivo General

Implementar algoritmos de manipulación de vectores numéricos utilizando JavaScript, incorporando nuevas funcionalidades al sistema base para optimizar el procesamiento y gestión de los datos.

### Objetivos Específicos

- Implementar algoritmos de ordenamiento que permitan organizar los elementos del vector de forma ascendente y descendente.
- Desarrollar métodos de búsqueda secuencial y binaria para localizar valores específicos dentro del vector.
- Incorporar herramientas de transformación que faciliten la modificación de los datos almacenados.
- Mejorar la interacción del usuario con el sistema mediante controles intuitivos y resultados visualizados en tiempo real.

## Desarrollo

Durante el desarrollo del proyecto se añadieron nuevos controles y botones a la interfaz de usuario con el propósito de ejecutar distintas operaciones sobre los vectores numéricos. Estas mejoras permitieron ampliar significativamente las funcionalidades del sistema original.

En primer lugar, se implementaron los algoritmos de ordenamiento por selección, burbuja e inserción, los cuales permiten organizar los elementos del vector tanto de manera ascendente como descendente. Estos métodos contribuyen a una mejor visualización y análisis de la información almacenada.

Asimismo, se desarrollaron las funciones de búsqueda secuencial y búsqueda binaria, facilitando la localización eficiente de valores específicos dentro del vector. La búsqueda binaria, en particular, optimiza el tiempo de búsqueda cuando los datos se encuentran previamente ordenados.

Adicionalmente, se incorporaron diversas herramientas de transformación de datos, entre las que destacan la inversión del vector, la eliminación de elementos duplicados, la rotación de posiciones y el intercambio de elementos específicos. Estas funcionalidades proporcionan una mayor flexibilidad en la manipulación de la información y permiten realizar diferentes operaciones de forma rápida y sencilla.
